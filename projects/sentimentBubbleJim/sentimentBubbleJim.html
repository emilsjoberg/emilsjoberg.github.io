This was a fun project to work on. We had a ton of Twitter data and the objective was to show sentiment breakout. Most of the time there is a simple bar chart showing the sentiment breakout of a topic. The weakness of that though is the audience doesn't get to see more granularly into each sentiment. For example, which topics were negative? What were the largest topics spoken about in a negative way? I thought this bubble chart showed different aspects of sentiment around Windows 8 really well because the viewer can see which topics were mentioned most on Twitter both as a whole as well as in each sentiment bin. They can also see the size of each sentiment bin.
